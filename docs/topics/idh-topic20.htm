Brownian motion

 

 

•In this mode, dots move either coherently (i.e. opposite to the motion of the chair) or with Brownian motion (i.e. with the same speed as the chair but in a random direction).

•The Coherence parameter specifies the fraction of the dots that move coherently. The remainder have Brownian motion.

 

ØThe category (coherent or Brownian motion) of each dot changes every frame
