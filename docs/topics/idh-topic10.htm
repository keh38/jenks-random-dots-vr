Random dot modes

 

 

 

 

Random dot movement

 

This screenshot of the configuration dialog highlights the controls used to specify how the random dots move.The two primary modes of motion are "Brownian" and "White Noise", each with their own set of parameters.

 

 

The lifetime parameters (red box) apply to both modes.

 

Parameter

Meaning

Lifetime (ms)

time dot is visible, rounded to integer number of frames. -1 = infinite lifetime

Deadtime (ms)

time dot is invisible after the lifetime expires, rounded to integer number of frames

Strobed

all lifetimes begin and end at the same time. During the deadtime, no dots are visible and the screen is black

 

Note that for the Vive Pro Eye headset the frame rate is 90 fps.
